# Sample Translation Chapter - Demonstrating Urdu Toggle

## English Content

This is a sample chapter that demonstrates the Urdu translation functionality. When you toggle the language using the button at the top, you will see the content in Urdu.

This chapter explains the core concepts of Physical AI and how it enables robots to interact with the physical world. The translation functionality allows users to switch between English and Urdu languages seamlessly.

## More English Content

Physical AI systems must handle real-world physics, uncertainty, and dynamic environments. This requires sophisticated algorithms and sensor integration to enable robots to navigate complex scenarios.

## Key Features Demonstrated

1. **Language Toggle Button**: A button at the top right of the page allows switching between English and Urdu
2. **Persistent Preference**: The selected language preference is saved in browser's local storage
3. **Bilingual Content**: Content can be provided in both languages for seamless switching
4. **Accessibility**: The toggle button includes proper ARIA labels for accessibility

This implementation provides a complete solution for multilingual content in the Physical AI & Humanoid Robotics textbook.
